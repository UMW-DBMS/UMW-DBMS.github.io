User Input Help

  1. Select District and DSD.
  2. Enter Supabase email and password.
  3. Click Verify Login.
  4. Enter GitHub token (PAT with repo write access).
  5. Select MWS_ID.
  6. Click Select Proposal, choose proposal type, fill form, then submit.
