Introduction

This guide explains how to use the system to explore data and maps in the dashboard.

Follow the steps below to navigate through the application.

Let’s dive in!.🚀

Zoom In and Zoom Out

Use the map zoom controls to adjust the map view.

  • Click the "+" button to zoom in
  • Click the "−" button to zoom out

Finding Your Data 🔍

To see specific information, follow this quick flow:

Select Location: Pick your District (Badulla, Kandy, or Nuwara Eliya) and your DSD.

Select a Watershed: Choose your MWS_ID from the list.

Don’t know the IDs?

Just click any spot on the map! A popup will instantly tell you the District, DSD, and MWS_ID for that exact location.

After selecting the District and DSD, choose the MWS_ID (Mini Watershed ID) from the dropdown list.

Change the Language

🌐 Language Note: You can switch the entire dashboard between English, Sinhala, or Tamil at any time using the language selector!

Change the Base Map

Click on the “OpenStreetMap” option in the bottom center ribbon to change the base map style.

Print the Map

Click the Print button to generate and print the map with its elements.

Learn About Dashboard Functions

Click the elements listed in the left-side section bar to learn how each feature works in the dashboard.

How to Access the Dashboard

Select the District:

Choose a district from the dropdown list: Badulla, Kandy, or Nuwara Eliya.

Select the DSD:

After selecting the district, choose the relevant DSD (Divisional Secretariat Division) from the available options.

Select the MWS_ID:

If you already know the MWS_ID, select it from the dropdown list.

If you do not know the MWS_ID, simply click on the map. A popup will appear displaying the MWS_ID along with the corresponding District and DSD information.

Proceed:

After completing all selections, click the “Go” button to load the data.

Base Layers

Click on the Base Layer button available on the map interface.

Choose the Required Layers

In the Base Layer panel, you will see several thematic categories such as:

  • Administration
  • Land Cover
  • Reservation
  • Topography
  • Soil
  • Risk and Hazard
  • Location
  • Demography
  • Hydrology
  • Water Features
  • Transportation

Enable Layers

Tick the checkbox next to the layer(s) you want to view.

View the Map

The selected layers will appear on the map, displaying the relevant information for the selected Mini Watershed (MWS_ID).

How to Access Proposals

Click the Proposal Button

After selecting the required Mini Watershed (MWS_ID), click the Proposal button to display the proposals related to that mini watershed.

View Proposal Locations

The proposals will appear in the layer panel on the left side.

To display the proposal locations on the map, tick the relevant checkboxes in the layer panel. The selected proposals will then be shown within the chosen Mini Watershed.

  • Soil and Water Conservation
  • Water Resource Management
  • Water Quality Improvement
  • Minor Tank Restoration

View Detailed Information

Click on any proposal point on the map to open a popup displaying detailed information about the selected proposal.

How to Use the Climate Layer Panel🌡️

Click on the Climate Layer button to access climate-related data layers for the selected project area.

Available Climate Data

The panel provides the following monthly datasets:

  • Rainfall
  • Maximum Temperature
  • Minimum Temperature
  • Evaporation
  • NDVI (Normalized Difference Vegetation Index)

Display Data on the Map

Tick the checkbox next to the desired dataset(s) to display the corresponding data layers on the map for the selected project area.

View Station Data on the Map

Click any point to view more information about the selected station.

How to View Station Data

Click the Station Data button to access the available station information.

View Station Data Layers

After clicking the button, the layer panel on the left side will display different types of station data, including:

  • Water Quality
  • Rainfall
  • Temperature
  • Evaporation

Display Stations on the Map

Tick the checkboxes next to the required station data layers in the layer panel to display the corresponding stations on the map.

View Station Information

Click on any station point on the map to open the detailed information about the selected station.

How to Access the Data Catalog

Click the Data Catalog Button and then will display the available datasets related to the selected Mini Watershed.

Download Data

From the catalog, you can select the required data files (geojson/shp) for the chosen Mini Watershed (MWS_ID) and download from the Download button.

Alternatively, you can click the 'Open MWS Data Catalog' button at the bottom right, Enter the MWS_ID, and download the data you need.

How to View Data in a Table Format

Click the Data Table button to access the tabular data view for the selected Mini Watershed.

You can download the table in CSV format.

How to View Statistics

Click the Statistics button to access statistical information related to the selected Mini Watershed.

Select a Category for View Graphs

Choose one of the following categories to view its statistics for the selected Mini Watershed (MWS_ID):

  • LULC (Land Use/Land Cover)
  • Hazard
  • Demography
  • Administration
  • Soil

Download Data

Click the 'Download CSV' button to export the data in a CSV format.

Click the 'Download Chart as image'button to download the graphical representation of the statistics.

How to Print a Map

Position the Map

Before printing, pan the map so that the required area appears below the “Select District” and “Select DSD” section, ensuring the selected Mini Watershed (MWS_ID) is centered on the screen.

Click the Print Button

Click the Print Map button to generate the map layout.

Print the Map

The system will produce a printable map with the necessary map elements, such as the legend, scale bar, and other relevant information, ready for printing or saving.

By zooming in or out, you can adjust the map view before printing.

To Download the Map click Download button with your preferred format:JPG or PNG

How to Measure Distance 📏

Click the Measure Distance button to activate the measurement tool.

First click on the map to place the starting point, then click again to place the ending point.

The distance will be displayed.

When you are done measuring, click the Measure Distance button again to deactivate the tool.

How to Add new Proposal data through Login

Click the Login Button

Click the Login button to access the login page.

Select District and DSD.

Choose the relevant District and DSD (Divisional Secretariat Division) from the dropdown menus.

Enter Login Credentials

Enter the Email and Password provided by the administrator.

Verify the Login

Click Verify Login button to proceed.

Enter GitHub Token

Enter the GitHub token provided by the Administrator.

Only authorized users with the email, password, and GitHub access token are allowed to upload new data.

Select MWS_ID

Choose the relevant MWS_ID (Mini Watershed) from the list

Select the Proposal Type

Select the appropriate proposal name from the following list:

  • DWS
  • LANDSLIDES
  • MI
  • MT
  • SR
  • SWC_ON
  • SWC_OFF
  • WP
  • WSP

Enter the Required Data

Fill in the necessary proposal details and information in the provided fields.

Upload a Photo (Optional)

If available, upload a photo related to the proposal.

Photo Size Requirement

Ensure that the photo file size is less than 1 MB before uploading.
