📖 Introduction
🗺️ How to Access the Dashboard

Select the District

Choose a district from the dropdown list: Badulla, Kandy, or Nuwara Eliya.

Select the DSD

After selecting the district, choose the relevant DSD (Divisional Secretariat Division) from the available options.

Select the MWS_ID

If you already know the MWS_ID, select it from the dropdown list.

If you do not know the MWS_ID, simply click on the map. A popup will appear displaying the MWS_ID along with the corresponding District and DSD information.

Proceed

After completing all selections, click the "Go" button to load the data.

🗂️ Base Layers

Click on the Base Layer button available on the map interface.

Choose the Required Layers

In the Base Layer panel, you will see several thematic categories:

  • Administration
  • Land Cover
  • Reservation
  • Topography
  • Soil
  • Risk and Hazard
  • Location
  • Demography
  • Hydrology
  • Water Features
  • Transportation

Enable Layers

Tick the checkbox next to the layer(s) you want to view.

View the Map

The selected layers will appear on the map, displaying the relevant information for the selected Mini Watershed (MWS_ID).

📋 How to Access Proposals

Click the Proposal Button

After selecting the required Mini Watershed (MWS_ID), click the Proposal button to display the proposals related to that mini watershed.

View Proposal Locations

The proposals will appear in the layer panel on the left side. Tick the relevant checkboxes to display proposal locations on the map.

  • Soil and Water Conservation
  • Water Resource Management
  • Water Quality Improvement
  • Minor Tank Restoration

View Detailed Information

Click on any proposal point on the map to open a popup displaying detailed information about the selected proposal.

🌡️ How to Use the Climate Layer Panel

Click on the Climate Layer button to access climate-related data layers for the selected project area.

Available Climate Data

The panel provides the following monthly datasets:

  • Rainfall
  • Maximum Temperature
  • Minimum Temperature
  • Evaporation

Display Data on the Map

Tick the checkbox next to the desired dataset(s) to display the corresponding data layers on the map for the selected project area.

View Station Data on the Map

Click any point to view more information about the selected station.

📡 How to View Station Data

Click the Station Data button to access the available station information.

View Station Data Layers

After clicking the button, the layer panel will display different types of station data including:

  • Water Quality
  • Rainfall
  • Temperature
  • Evaporation

Display Stations on the Map

Tick the checkboxes next to the required station data layers to display the corresponding stations on the map.

View Station Information

Click on any station point on the map to open the detailed information about the selected station.

📦 How to Access the Data Catalog

Click the Data Catalog button — this will display the available datasets related to the selected Mini Watershed.

Download Data

From the catalog, you can select the required data files (geojson/shp) for the chosen Mini Watershed (MWS_ID) and download via the Download button.

💡 Alternatively, click the 'Open MWS Data Catalog' button at the bottom right, enter the MWS_ID, and download the data you need.
📊 How to View Data in Table Format

Click the Data Table button to access the tabular data view for the selected Mini Watershed.

You can download the table in CSV format using the export option provided.

📈 How to View Statistics

Click the Statistics button to access statistical information related to the selected Mini Watershed.

Select a Category to View Graphs

Choose one of the following categories to view its statistics for the selected Mini Watershed (MWS_ID):

  • LULC (Land Use/Land Cover)
  • Hazard
  • Demography
  • Administration
  • Soil

Download Data

Click 'Download CSV' to export the data in CSV format.

Click 'Download Chart as Image' to download the graphical representation of the statistics.

🖨️ How to Print a Map

Click the Print Button

Click the Print Map button to generate the map layout.

Print the Map

The system will produce a printable map with a legend, scale bar, and other relevant elements — ready for printing or saving.

Zoom in or out to adjust the map view before printing.

💾 Click the Download button to save the map in your preferred format: JPG or PNG.
📏 How to Measure Distance

Click the Measure Distance button to activate the measurement tool.

Click on the map to place the starting point, then click again to place the ending point.

The distance will be displayed on screen.

ℹ️ When you are done measuring, click the Measure Distance button again to deactivate the tool.
🔐 How to Add New Proposal Data via Login

Click the Login Button

Click the Login button to access the login page.

Select District and DSD

Choose the relevant District and DSD (Divisional Secretariat Division) from the dropdown menus.

Enter Login Credentials

Enter the Email and Password provided by the administrator.

Verify the Login

Click the Verify Login button to proceed.

Enter GitHub Token

Enter the GitHub token provided by the Administrator.

🔒 Only authorized users with a valid email, password, and GitHub access token are allowed to upload new data.

Select MWS_ID

Choose the relevant MWS_ID (Mini Watershed) from the list.

Select the Proposal Type

  • DWS
  • LANDSLIDES
  • MI
  • MT
  • SR
  • SWC_ON
  • SWC_OFF
  • WP
  • WSP

Enter the Required Data

Fill in the necessary proposal details and information in the provided fields.

Upload a Photo (Optional)

If available, upload a photo related to the proposal.

📷 Ensure the photo file size is less than 1 MB before uploading.
💧 How to Update Water Quality Data via Login

Click the Water Quality Button

Click the Water Quality button and open the Water Quality page.

Select the Agency

Choose the relevant agency from the available options: CEA, MASL, or NWSDB.

  • CEA
  • MASL
  • NWSDB

Enter Login Credentials

Enter the authorized Email, Password, and GitHub token provided for data updates.

Select the CSV Name

After selecting the agency, choose the required CSV file name from the dropdown list.

Load the Existing File

Click Load Existing CSV to view and download the current file already available in the dashboard repository.

Upload the Updated CSV

Select the updated CSV file from your computer and click Upload CSV to replace the existing dataset.

🔒 Only approved agency users can access and update water quality CSV files.
🗂️ How to Update LULC Data via Login

Click the LULC Button

Click the LULC button and open the LULC page.

Select District, DSD, and MWS_ID

Choose the relevant District, DSD, and MWS_ID to identify the area for the upload.

Enter Login Credentials

Enter the authorized Email and Password, then provide the GitHub token with access to the repository.

Load Existing LULC Files

Click Load LULC Files to display the currently available LULC files linked to the selected watershed.

Select the GeoJSON File

Choose the updated LULC GeoJSON file from your computer before uploading.

Upload the LULC Data

Click Upload GeoJSON to save the updated LULC file to the dashboard repository.

ℹ️ Use the correct MWS_ID and file before uploading so the selected watershed data is updated in the right location.
